# Red-Star 阳光学情分析系统 - 测试指南

## 系统架构

- **后端**: PHP 8.3 + SQLite 3
- **前端**: 原生 JavaScript（零框架、零构建）
- **特点**: 零部署、零编译、修改即生效

## 快速启动

### 1. 确保 PHP 环境

- PHP 8.3+ 已安装
- 启用扩展：pdo, pdo_sqlite, json, mbstring, openssl
- Nginx 已配置伪静态转发 `/api/*` 到 `api/index.php`

### 2. 访问系统

- 登录页：`http://rs.cn/login.php`
- 主页：`http://rs.cn/index.php`

## 常见问题

### 1. 页面空白

- 检查浏览器控制台是否有 JS 错误
- 确认 `currentUser` 变量已正确注入

### 2. API 返回 404

- 检查 Nginx 伪静态配置
- 确认 `/api/*` 转发到 `api/index.php`

### 3. 登录失败

- 确认数据库已初始化（首次访问自动创建）
- 检查 `data/redstar.db` 文件是否存在

### 4. 大学数据为空

- 访问 `http://rs.cn/api/import_data_web.php` 导入数据

## 文件结构

```
rs.cn/
├── api/
│   ├── index.php          # API 路由入口
│   ├── config.php         # 配置中心
│   ├── schema.sql         # 数据库表结构
│   ├── import_universities_2025.sql  # 大学数据 SQL
│   └── import_data_web.php          # Web 导入脚本
├── assets/
│   ├── style.css          # 全局样式
│   └── app.js             # 前端主逻辑
├── data/
│   └── redstar.db         # SQLite 数据库
├── backups/               # 备份目录
├── index.php              # 主页面
└── login.php              # 登录页
```

## 技术特点

1. **零依赖** - 无需 Node.js、Composer、npm
2. **所见即所得** - 修改 PHP/JS/CSS 文件，刷新浏览器即生效
3. **轻量** - 整个项目约 2-5 MB
4. **便携** - 打包拷贝到新服务器即可运行
